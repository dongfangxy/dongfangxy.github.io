Simulink Control Design

Below is a list of new features or enhancements I contributed to Simulink Control Design at MathWorks.

Disclaimer: This page is not part of the official release notes or any official documentation of Simulink Control Design. Information given below is quoted from the publicly accessible SCD Release Notes. If there is any discrepency, please refer to the official release notes. .

R2017a

  • Custom Constraints and Objective Functions for Trimming Simulink Models: Calculate operating points with increased flexibility [See more]

  • Bounds on State Derivatives During Trimming: Constrain derivatives of model states that are not at steady state [See more] [Doc:operspec]

  • addoutputspec Command: Add output specification to multiple operating point specification objects in an array [Doc:addoutputspec]

R2016b

  • Batch Trimming for Parameter Variation: Vary model parameters, and compute multiple operating points using a single model compilation [See more]

  • Improved LPV System Construction: Compute operating point offsets for model inputs, outputs, states, and state derivatives during linearization [See more]

  • operspec Command: Create array of operating point specification objects [See more]
